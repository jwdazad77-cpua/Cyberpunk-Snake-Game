# Cyberpunk Snake Game

A modern cyberpunk-themed Snake game built with TypeScript, Vite, and full-stack architecture.

## Features
- Neon cyberpunk visual theme
- Responsive controls
- Client-server architecture
- Modular shared constants

## Tech Stack
- TypeScript
- Vite
- Node.js
- Express (if used)
- Modern frontend tooling

## Installation

```bash
npm install
npm run dev
